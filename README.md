## 0. Что такое .NET и CLI-команды `dotnet`

**.NET** — это бесплатная кроссплатформенная платформа от Microsoft для создания приложений. Она включает в себя среду выполнения (Runtime), которая запускает ваш код, и библиотеки.

Управление проектами происходит через консольный интерфейс — **C# CLI** (Command Line Interface). Все команды начинаются со слова `dotnet`.

### Основные терминальные команды

Выполняйте эти команды в терминале, находясь в папке вашего проекта:

*   `dotnet new console` — Создать новый пустой проект консольного приложения в текущей папке.
*   `dotnet build` — Скомпилировать (собрать) проект. Проверяет код на ошибки и создает исполняемый файл в папке `bin/`.
*   `dotnet run` — Скомпилировать и сразу запустить проект. Самая частая команда при разработке.
*   `dotnet watch` — Запустить проект в режиме «живой перезагрузки». Программа сама перезапустится, как только вы сохраните изменения в коде.
*   `dotnet clean` — Очистить все временные файлы сборки (удаляет папки `bin/` и `obj/`). Полезно, если проект ведет себя странно и «глючит».

### Структура проекта (что создают эти команды)

Когда вы пишете `dotnet new console`, C# создает несколько важных файлов:
*   `Program.cs` — Главный файл, где пишется ваш код.
*   `ИмяПроекта.csproj` — Главный файл настроек проекта (XML-формат). Там указана версия .NET и подключенные сторонние библиотеки. Никогда не удаляйте его.

## 1. Структура проекта и консоль

Минимальный шаблон программы и базовый ввод-вывод данных.

```csharp
using System; // Подключение пространства имен

namespace ConsoleApp {
    class Program {
        static void Main(string[] args) {
            // Вывод текста с переносом строки и без
            Console.WriteLine("Hello World!"); 
            Console.Write("Введите что-нибудь: ");

            // Чтение строки из консоли
            string input = Console.ReadLine(); 
            Console.WriteLine("Вы ввели: " + input);

            // Ожидание нажатия одной клавиши (чтобы консоль не закрывалась)
            Console.ReadKey(); 
        }
    }
}
```

## 2. Типы данных и переменные

В C# жесткая типизация. Каждому типу выделяется строгое количество байт в памяти.

### Целочисленные (Integer)

| Тип | Размер | Диапазон / Описание |
| :--- | :--- | :--- |
| `byte` | 1 байт | от 0 до 255 |
| `short`| 2 байта | от -32 768 до 32 767 |
| `int` | 4 байта | от -2.14 млрд до 2.14 млрд (основной тип) |
| `uint` | 4 байта | от 0 до 4.29 млрд (только положительные) |
| `long` | 8 байт | огромные целые числа |
| `ulong`| 8 байт | огромные только положительные целые числа |

### Числа с плавающей точкой (Дробные)
*   `float`: 4 байта. Требует суффикс `f` при объявлении (`1.6f`).
*   `double`: 8 байт. Самый используемый тип для дробей (`32.12`).

### Символы, Текст и Логика
*   `char`: Один символ в одинарных кавычках (`'A'`).
*   `string`: Строка (список символов) в двойных кавычках (`"name"`).
*   `bool`: Логический тип, принимает только `true` или `false`.

### Преобразование типов и dynamic
```csharp
// Конвертация строки в число
int n1 = Convert.ToInt32(Console.ReadLine());
double n2 = double.Parse(Console.ReadLine());

// dynamic — тип, меняющий свой тип на лету во время выполнения
dynamic dyn = 0;      // Сейчас это int
dyn = "name";         // Теперь это string
Console.WriteLine(dyn.ToUpper()); // Выведет: NAME
```

## 3. Арифметика и класс Math

### Операторы и инкремент
```csharp
double result = (v + 10d - 10) * 2 / 10 % 1.23; // % — остаток от деления

result += 5; // Сокращенная запись (result = result + 5)

// Инкремент и декремент
result++; // Постфиксный (увеличение после использования)
++result; // Префиксный (увеличение до использования)
```

### Класс Math (Встроенные математические функции)
```csharp
double pi = Math.PI;          // Число Пи (~3.14159)
int abs = Math.Abs(-10);      // Модуль числа (10)

Math.Ceiling(4.11);           // Округление вверх (= 5)
Math.Floor(4.99);             // Округление вниз (= 4)
Math.Round(4.5);              // Округление по правилам математики (= 4 или 5)

// Также доступны: Math.Sin(), Math.Cos(), Math.Sqrt() (корень), Math.Pow() (степень), Math.Min(), Math.Max()
```

### Форматирование и интерполяция строк (Плейсхолдеры)
```csharp
// Подстановка переменных по индексам {0}, {1}
Console.WriteLine("Формат: {0} и {1}", "первый", "второй");
// Самый современный способ форматирования строк в C#.
Console.WriteLine($"Еще один {"Cпособ".ToLower()}");
```

## 4. Условные операторы (if / switch)

### Конструкция if-else и Логические И/ИЛИ
```csharp
int a = int.Parse(Console.ReadLine());

if (a == 1) {
    Console.WriteLine("a равно 1");
} else if (a > 1) {
    Console.WriteLine("a больше 1");
} else {
    Console.WriteLine("a меньше 1");
}

bool b = false;
// && — Логическое И (AND)
// || — Логическое ИЛИ (OR)
// !  — Отрицание НЕ (NOT)
if ((b || !b) && (a > 1)) {
    Console.WriteLine("Условие выполнилось");
}
```

### Конструкция switch-case
Удобна, когда нужно перебирать конкретные значения одной переменной.
```csharp
switch (a) {
    case 1:
        Console.WriteLine("Ветка 1");
        break; // break обязателен!
    case 0:
        Console.WriteLine("Ветка 0");
        break;
    default:
        Console.WriteLine("Если ни один case не подошел");
        break;
}
```

## 5. Массивы (Одномерные и Многомерные)

Массивы имеют **фиксированную** длину, которую нельзя изменить после создания.

### Одномерные массивы
```csharp
// Способ 1: Выделение памяти, заполнение по индексам
byte[] aboba = new byte[4]; // Массив на 4 элемента (индексы 0, 1, 2, 3)
aboba[0] = 1;
aboba[1] = 2;

// Способ 2: Инициализация сразу с элементами
string[] names = new string[] { "aboba", "goida" };

// Перебор массива через обычный цикл for
for (int i = 0; i < names.Length; i++) {
    Console.WriteLine(names[i]);
}
```

### Генерация случайных чисел в массив
```csharp
Random random = new Random();
short[] numbers = new short[10];

for (int i = 0; i < numbers.Length; i++) {
    numbers[i] = Convert.ToInt16(random.Next(5, 15)); // Числа от 5 до 14 включительно
}
```

### Двумерные (Многомерные) массивы `[,]`
```csharp
// Матрица из 2 строк и 3 колонок
char[,] symbols = new char[2, 3]; 

// Инициализация со значениями напрямую:
int[,] nums = {
    {1, 2, 3}, // Строка 0
    {4, 5, 6}, // Строка 1
    {7, 8, 9}  // Строка 2
};

// Вывод двумерного массива на экран через вложенные циклы
for (int i = 0; i < 2; i++) {
    for (int j = 0; j < 3; j++) {
        Console.Write(symbols[i, j] + " ");
    }
    Console.WriteLine(); // Перенос на новую строку после завершения ряда
}
```

## 6. Цикл foreach

Используется для быстрого и безопасного чтения всех элементов коллекции. Изменять элементы коллекции внутри `foreach` **нельзя**.

```csharp
short[] fornums = new short[] { 1, 2, 3, 4 };

foreach (short num in fornums) {
    Console.WriteLine(num); // num — копия текущего элемента
}

// Перебор многомерного массива через foreach разворачивает его в одну сплошную линию
foreach (short num in nums) {
    Console.WriteLine(num); // Выведет последовательно: 1, 2, 3, 4, 5...
}
```

## 7. Динамические списки (List)

Списки находятся в пространстве `using System.Collections.Generic;`. В отличие от массивов, они могут динамически расширяться (менять размер).

```csharp
// Создание списка целых чисел
List<int> list = new List<int>() { 1, 2, 3 };

list.Add(4);       // Добавит 4 в конец списка -> [1, 2, 3, 4]
list.Remove(1);    // Удалит ПЕРВОЕ встреченное значение "1" -> [2, 3, 4]
list.RemoveAt(0);  // Удалит элемент по ИНДЕКСУ 0 (это двойка) -> [3, 4]

// Быстрый вывод списка на экран
foreach (int item in list) {
    Console.Write(item + " ");
}

// Полезные встроенные методы:
// list.Sort();    // Сортировка по возрастанию
// list.Reverse(); // Перевернуть список задом наперед
// list.Count;     // Получить текущее количество элементов (аналог .Length у массивов)
```

## 8. Функции (Методы) и Перегрузка

Функции в C# пишутся внутри классов. Для использования в `Main` без создания объектов класса они должны иметь модификатор `static`.

```csharp
class Program {
    static void Main(string[] args) {
        Console.WriteLine(Aboba());        // Выведет: string
        Console.WriteLine(Aboba("test"));  // Выведет: testtest

        byte[] array = { 1, 2, 3 };
        Console.WriteLine(Summa(array));   // Выведет: 6
    }

    // public — метод доступен везде
    // static — принадлежит самому классу, а не его экземплярам
    // string — тип возвращаемого значения (void, если ничего не возвращает)
    public static string Aboba() {
        return "string";
    }

    // ПЕРЕГРУЗКА МЕТОДА: имя то же, но другие входные аргументы
    public static string Aboba(string arg) {
        return arg + arg;
    }

    // Передача массива в качестве аргумента
    public static int Summa(byte[] array) {
        int res = 0;
        foreach (int elem in array) {
            res += elem;
        }
        return res;
    }
}
```

## 9. Строки и Работа с Файлами (I/O)

### Полезные методы строк (String)
Строка — это неизменяемый (immutable) массив символов `char`.

```csharp
string s = "aboba";
Console.WriteLine(s[2]); // Доступ по индексу -> 'o'

string people = "a, b, o";
// Разделение строки в массив по разделителю
string[] parts = people.Split(", "); 
// Сборка массива обратно в строку с новым разделителем -> "a/b/o"
Console.WriteLine(String.Join("/", parts)); 

Console.WriteLine("   3   0   ".Trim()); // Удаляет пробелы в начале и конце -> "3   0"
Console.WriteLine("1234".Substring(2, 2)); // Вырезает подстроку (с какого индекса, сколько символов) -> "34"

// s.ToUpper() — в верхний регистр, s.ToLower() — в нижний
```

### Запись и Чтение файлов через FileStream
Использование конструкции `using` автоматически закрывает файл и освобождает память (стрим) после выхода из блока `{}`.

```csharp
using System.IO;
using System.Text;

// ЗАПИСЬ: Создает или перезаписывает файл text.txt
using (FileStream stream = new FileStream("text.txt", FileMode.OpenOrCreate)) {
    // Перевод строки в массив байт в стандартной кодировке
    byte[] array = Encoding.Default.GetBytes("Данные для записи");
    stream.Write(array, 0, array.Length);
}

// ЧТЕНИЕ: Безопасный способ вычитать файл целиком
using (FileStream stream = File.OpenRead("text.txt")) {
    byte[] array = new byte[stream.Length];
    
    // .ReadExactly гарантирует, что буфер заполнится полностью, исключая баги незавершенного чтения
    stream.ReadExactly(array, 0, array.Length);

    string text = Encoding.Default.GetString(array);
    Console.WriteLine(text);
}
```

## 10. Обработка Исключений (Exceptions)

Используется для защиты программы от аварийного завершения (падения) при ошибках (например, неверный ввод пользователя).

```csharp
int a = -1;
try {
    // Код, который может вызвать ошибку
    a = Convert.ToInt16(Console.ReadLine());
} 
catch (FormatException ex) {
    // Сработает ТОЛЬКО если пользователь ввел буквы вместо числа
    Console.WriteLine("Вы ввели не число!");
}
catch (Exception e) {
    // Сработает на любую другую ошибку. e.Message — текст системной ошибки
    Console.WriteLine(e.Message);
} 
finally {
    // Этот блок выполнится ВСЕГДА (была ошибка или нет). Удобно для очистки ресурсов
    Console.WriteLine("Блок try-catch завершен");
}
```
